# DocePreço MVP

Aplicativo web para confeiteiras calcularem orçamentos de bolos, docinhos gourmet e combos.

## Como rodar

```bash
npm install
npm run dev
```

Depois abra o endereço mostrado no terminal.

## O que já tem

- Dashboard com total de orçamentos, receita média, lucro estimado e próximas entregas
- Cadastro de clientes
- Cadastro de ingredientes com cálculo automático de custo unitário
- Cadastro de receitas/produtos reutilizáveis
- Orçamento de bolo, docinhos e combo
- Cálculo de custo, preço sugerido e lucro
- Histórico de pedidos com status
- Configurações da confeitaria
- Salvamento no navegador via localStorage
